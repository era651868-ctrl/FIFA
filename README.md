## 🛠️ Step-by-Step Installation & Setup

Follow these commands to deploy and test the matrix engine locally:

### 1. Clone the Architecture Repository
```bash
git clone [https://github.com/era651868-ctrl/FIFA.git](https://github.com/era651868-ctrl/FIFA.git)
cd FIFA

```
### 2. Install Dependencies
```bash
pip install -r requirements.txt

```
### 3. Configure Workspace Credentials
Set your Google AI Studio token as an environment variable:
```bash
# On Linux/macOS
export GEMINI_API_KEY="your_actual_api_key_here"

# On Windows (Command Prompt)
set GEMINI_API_KEY="your_actual_api_key_here"

# On Windows (PowerShell)
$env:GEMINI_API_KEY="your_actual_api_key_here"

```
### 4. Execute the Application Local Host Instance
```bash
streamlit run app.py

```
